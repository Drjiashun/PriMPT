Data Description

This directory contains the data files used for PriMPT, a prior-informed multi-scale paired-token representation learning framework for CRISPR/Cas9 off-target activity classification.

The data are organized into raw datasets and processed train/validation/test splits. All datasets are used only for academic research and reproducibility purposes.

Raw Data Sources

Five public CRISPR/Cas9 guide–target activity datasets were used in this project:

K562
HEK293T
II4
II5
CRISPOR

The datasets were obtained from the supplementary data of the following CRISPR/Cas9 benchmarking study:

Zhang, G.; Luo, Y.; Dai, X.; Dai, Z. Benchmarking deep learning methods for predicting CRISPR/Cas9 sgRNA on- and off-target activities. Briefings in Bioinformatics 2023, 24(6), bbad333. DOI: 10.1093/bib/bbad333.

The original dataset sources are listed below.
K562 and HEK293T, Chuai et al., Genome Biology, 2018; II4 and II5, Lin et al., Advanced Science, 2020; CRISPOR, Peng et al., Bioinformatics, 2018.

Data Redistribution Notice

The datasets included in this directory were derived from previously published public datasets and benchmark resources. We do not claim ownership of the original experimental data.

The data are provided only for academic research and reproducibility purposes. Users should check and follow the license, terms of use, and data redistribution policy of the original data sources.

If redistribution of any original dataset is restricted by its original license or data-use policy, users should obtain the raw data from the original publications or benchmark source and then run the preprocessing scripts provided in this repository.
